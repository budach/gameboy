# Basic Game Boy Emulator

## Build

```bash
make release
```

## Run

```bash
./gameboy <gb_rom_file>
```

## Controls

| Game Boy | Keyboard   |
|----------|------------|
| D-pad    | Arrow keys |
| A        | A          |
| B        | S          |
| Start    | Enter      |
| Select   | Backspace  |

## Other controls

| Action                | Key      |
|-----------------------|----------|
| Increase FPS by 30    | Page Up  |
| Decrease FPS by 30    | Page Down |
